## Sobre o projeto 💬

O **Bateria** é uma aplicação WEB Playback de bateria, podendo criar sua própria composição ao pressionar as teclas

## Funcionalidades 🧠

A aplicação contém as seguintes funcionalidades:

### Index.html 📕

- Criar sua Própria Composição
